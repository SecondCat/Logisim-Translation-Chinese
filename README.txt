为了使Logisim变得简单易用，我将Logisim需要的 Java JRE 和 logisim.jar 封装起来，使得 Logisim 可以在没有配置 JRE环境 的 Windows操作系统系统 中运行。

添加的JRE版本为JDK-15.0.1。
LogisimLauncher.exe 是用C语言编写的链接启动器，LogisimLauncher.c是它的源代码。
logisim.jar是以原作者 Carl Burch 发布的 logisim2.7.1 （下载地址：https://sourceforge.net/projects/circuit/）为基础，增加中文语言资源得到的，因此该软件支持中文语言。

支持：Windows11，Windows 10，Windows 8，Windows 7

Logisim官方网址：http://www.cburch.com/logisim/

To make Logisim easy to use, I have wrapped the Java JRE and logisim.jar that Logisim needs so that Logisim can run on Windows OS systems that do not have a JRE environment configured.

The added JRE version is JDK-15.0.1.
"LogisimLauncher.exe" is a linked launcher written in C, and "LogisimLauncher.c" is its source code.
"logisim.jar" is based on logisim2.7.1 (download at https://sourceforge.net/projects/circuit/) released by the original author Carl Burch, and add Chinese language resources to get it, so the software supports Chinese language.

Support: Windows11, Windows 10, Windows 8, Windows 7

Logisim official website: http://www.cburch.com/logisim/
